About

This is a miniature project - 'PUNARJANI' done as a part of 'Build From Home' , an initiative led by Tinkerhub. The main aim of this program is to minimize the effort on finding available vaccination centers , slots when one register for vaccination. Mr PJ ,the bot runs on discord platform. This is a user friendly bot which acts as an assistant of covid vaccination system. This is an open source program which means one can contribute more features or build new versions of the same. We believe that everyone must get equal opportunity and equal care when it comes to critical sectors like health and education. We may not be able to solve the vaccine shortage but we can definitely be part of the solution.
